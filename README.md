# 📦 E_Commerce_Platform

## 🛍️ Full-Stack Multivendor E-commerce Platform

This repository contains a production-ready, multivendor e-commerce solution featuring mobile apps for customers and vendors, a web-based admin panel, and a powerful backend API. Built using **Flutter**, **Node.js**, **Express.js**, **Firebase**, **MongoDB**, and automated with **Codemagic CI/CD**, this platform is ideal for launching a scalable e-commerce business.

---

## 📁 Project Structure
  E_Commerce_Platform/
  
  ├── ecommerce_backend/ # Node.js + Express.js REST API
  
  ├── ecommerce_store/ # Customer mobile app (Flutter)
  
  ├── ecommerce_vendor_app/ # Vendor mobile app (Flutter)
  
  ├── ecommerce_web_admin/ # Web admin panel (Flutter web)
  
  ├── .gitignore
  
  └── codemagic.yaml # Codemagic CI/CD pipeline

## ⚙️ Tech Stack
  | Layer      | Technology                                     
  |------------|------------------------------------------------
  | Frontend   | Flutter (Dart)                      
  | Backend    | Node.js, Express.js                 
  | Database   | Firebase (Authentication), MongoDB (Storages)
  | CI/CD      | Codemagic                           
  | Hosting    | Render

## 🚀 Key Features
  ### 👥 User Roles
  - **Customers** – Browse, search, and purchase products
  - **Vendors** – Manage inventory, orders, and analytics
  - **Admins** – Control platform operations, users, and content

  ### 🔧 Platform Features
  - 🔐 Secure authentication (JWT, Firebase Auth)
  - 🛍️ Product listings with images, categories & tags
  - 💳 Checkout with cart, address, and payment (extendable)
  - 📦 Order tracking & status updates
  - 📊 Admin dashboard with analytics
  - 📁 Cloud storage for product images
  - 📱 Mobile-first design (Android/iOS)
  - 🌐 Flutter Web for admin

## 🧱 CI/CD with Codemagic
  The project includes a codemagic.yaml configuration for automated builds and deployments.
  
  ✅ Auto-build Flutter apps
  
  ✅ Run tests and static analysis
  
  ✅ Optionally publish to stores or Firebase Hosting
